## Apache Hive Search 

✅ __Clone project__
```bash
git clone https://github.com/ezynook/hive-query.git
```
✅ __Deploy__
```bash
docker-compose up -d  --build
```
_สามารถเข้าไปเปลี่ยน Port ได้ที่ ```docker-compose.yml```_
